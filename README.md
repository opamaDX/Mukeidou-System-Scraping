# Mukeidou-System--Scraping
夢形堂の業務改善システムを作成するリポジトリ。スクレイピング➝CSVを作成

# 編集画面（admin.html）について

## 機能一覧

- ウォッチ数が0なら赤色、1~5は黒色、6~は緑色に変更
- 管理番号をクリックすると、コピーされた状態になる
- 商品名をクリックすると、商品詳細ページに遷移する
- チェックボックスをクリックすると赤色、青色に変更

    赤色のとき未チェック、青色のときはチェック済み

- 入力フォームには、価格改定ルールに則った価格を表示
- 増減率が絶対値30以上のとき、フォームを赤色に表示
